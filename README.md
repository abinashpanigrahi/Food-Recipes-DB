
# Food Recipes Web App

- This web application has 'Recipe of the Day' feature  

- Users can search a recipe by food name

- Displays some popular latest meals


## Deployed Link

- [Food Recipes App](https://quirky-goldberg-a1edbd.netlify.app/)
